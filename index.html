Variables

Variables

Las variables son espacios de memoria en la computadora donde se pueden almacenar datos de diferentes tipos.

Una variable es como una pequeña caja en la memoria de la computadora en la que podemos almacenar un valor.

A esa caja le ponemos un nombre (la "etiqueta") para poder referirnos a ese valor más tarde y es importante saber que el valor que contiene esa caja puede cambiar.

Tipos de variables

En Javascript tenemos 3 tipos de variables:

  • var
  • let
  • const

 

var es una palabra reservada que se utiliza para indicarle a Javascript que vamos a declarar una variable:

var nombreVariable;

let es una palabra reservada que se utiliza para indicarle a Javascript que vamos a declarar una variable  pero que solo se puede utilizar dentro de determinados bloques de código (generalmente limitados por llaves { } ) :

let nombreVariable;

const es una palabra reservada que se utiliza para indicarle a Javascript que vamos a declarar una variable CONSTANTE, es decir, que una vez que, además de existir solo dentro de ciertos bloques de código (como let), cuando le asignemos un valor, no podrá cambiar ese valor.

const nombreVariable;

Declaración y asignación de variables

Declaración de variables: 

var nombre;

var edad;

var apellido;

var presenteoNo;

var estatura;

Aquí sólo declaramos las variables. 

Asignación  de variables: 

Asignación de variables: utilizamos un operador de asignación, el signo de igual: =

Este operador no es de comparación sino de asignación. 

var nombre = "María";

var edad = "30";

var apellido = "Pérez";

var presenteoNo = true;

var estatura = 1,67;

¿Qué son los tipos de datos?

Los tipos de datos son clasificaciones que indican la naturaleza de la información que una variable puede contener.

JavaScript reconoce varios tipos de datos fundamentales, cada uno con características específicas.

En JavaScript no es obligatorio declarar el tipo de dato de una variable cuando la creas (es un lenguaje de tipado dinámico).

Solo indicas que vas a crear una variable (let, const, var), pero no especificas si será un número, texto, booleano, etc.
El tipo se asigna automáticamente según el valor que le pongas, y puede cambiar en tiempo de ejecución.

Tipos de datos

Tipos de datos más comunes en JavaScript:

Number (Números)
 Para cualquier valor numérico, ya sean enteros o con decimales.

Ejemplo: 

let edad = 15;       // Número entero

let estatura = 1.75; // Número con decimales

String (Cadenas de Texto)
Para caracteres alfanuméricos. Texto, palabras, frases. Siempre van encerrados entre comillas simples (') o dobles (").

Ejemplo: 

let nombre = "Ana";

let saludoAna = '¡Bienvenidos!';

Boolean (Booleanos)
Representan valores lógicos: true (verdadero) o false (falso).

Son fundamentales para las decisiones.

Ejemplo: 

let llueve = true;

let haySol = false;

Array (Arreglos/Listas)
Son colecciones ordenadas de elementos. Por ejemplo en una lista de frutas. Se definen con corchetes [] y los elementos se separan por comas.

Ejemplo: 

 let tesorosEncontrados = ["manzana", "banana", "naranja", "pera"];

let numerosDeFrutas = [1, 7, 13, 20];

Object (Objetos)
Son colecciones de propiedades, donde cada propiedad tiene un nombre (clave) y un valor. Permiten representar entidades más complejas, como por ejemplo la descripción de un jugador.

Se definen con llaves { }.

Ejemplo: 

let jugador = {  

nombre: "Juan",  

edad: 14,  

rango: "Principiante",

tieneMedalla: true };

Operadores

¿Qué son los operadores?

Los operadores son símbolos especiales que le dicen a JavaScript que realice alguna operación con uno o más valores (llamados operandos).

Los operadores son símbolos o palabras clave que se utilizan para realizar operaciones sobre uno o más valores (o variables), produciendo un nuevo valor.

Permiten realizar cálculos matemáticos, comparaciones lógicas, asignaciones y muchas otras acciones.

Tipos de operadores

Operadores aritméticos: Para realizar cálculos matemáticos. Ejemplo: 

let a = 15;

let b = 12;

console.log(a + b); // Resultado: 27

console.log(a - b); // Resta: 3

console.log(a * b); // Multiplicación: 180

console.log(a / b); // División: 1.25

console.log(a % b); // Módulo (resto de la división): 3

 

Operadores de asignación

Se usan para asignar un valor a una variable. El más común es =, pero hay 0tros. Por ejemplo: 

let puntaje = 100; // Asigna 100 a puntaje
puntaje += 50; // Es lo mismo que: puntaje= puntaje+ 50; (ahora es 150)
puntaje -= 20; // Es lo mismo que: puntaje= puntaje- 20; (ahora es 130)

Operadores lógicos
Se usan para combinar o negar condiciones booleanas.

let llueve = true;
let nublado= false;

// AND (&&): Verdadero solo si ambas condiciones son verdaderas
console.log(llueve&& nublado); // false (porque nublado es false)

// OR (||): Verdadero si al menos una condición es verdadera
console.log(llueve || nublado); // true (porque llueve es true)

// NOT (!): Invierte el valor booleano
console.log(!llueve); // false

Operadores de comparación

Se utilizan para comparar dos valores.

El resultado de una comparación siempre es un valor booleano: true o false.

(Abarcan todos los operadores que comparan valores, no importa si son de igualdad o de relación).
Ejemplo: ==, ===, !=, !==, >, <, >=, <=.
Los operadores relacionales son un subconjunto de los de comparación, usados específicamente para relaciones de orden (mayor, menor, mayor o igual, menor o igual).
Ejemplo: >, <, >=, <=.

Ejemplo:

Se recomienda usar === y !== en lugar de == y != para evitar errores por conversión automática de tipos (coerción).
Por ejemplo: 0 == false es true, pero 0 === false es false.
